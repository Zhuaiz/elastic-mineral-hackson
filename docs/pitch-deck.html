Nº 02THE PROBLEM
为什么矿物鉴定难

单一模态，都不够。

只看照片 → 混淆
玛瑙 · 玉髓 · 碧玉
都是隐晶质石英，肉眼与图像模型都难分。
只看属性 → 不收敛
硬度7 · 白条痕 · 玻璃光泽
被几十种矿物共享，定不到唯一种。

几千种矿物，很多长得几乎一样、属性也高度重叠。任何单一信号都不足以定种。

Nº 03THE IDEA
我们的答案

把三种信号
融合进一次检索。

每个标本在 Elasticsearch 里同时可被三种方式检索——图像向量、属性全文、结构化字段。 ES 9.3 的 rrf retriever 把它们在一个查询里倒数排名融合。这是 Elastic 原生能力，不是我们拼的。

RRF

候选矿物

mineralogist agent

我给它两个 ES|QL 属性工具和一个融合检索工具，它自主编排：先按硬度筛，还是先按图搜。

Nº 07LIVE DEMO · 拍照识矿
让屏幕说话

三栏并排：
只看图 · 只看字 · RRF 融合。

拖入一张标本照片 → 只看图，玛瑙玉髓碧玉混在一起；补上属性和硬度过滤 → 融合这一栏第一名立刻锁定。 这一栏赢，就是 RRF 的价值。
